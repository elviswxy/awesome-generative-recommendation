# Awesome Generative Recommendation

A curated, practical, and research-oriented repository for the next generation of recommendation systems. 

This repository brings together datasets, papers, tutorials, benchmarks, and practical guides for building generative recommendation systems. Our goal is to provide a unified resource center for researchers and engineers exploring how modern generative technologies can reshape retrieval, ranking, and personalized recommendation.

## Contributing

Contributions are what make the open-source community such an amazing place to learn, inspire, and create. Any contributions you make are **greatly appreciated**.

### How to Contribute

1. **Fork** the Project.
2. **Create** your Feature Branch (`git checkout -b feature/AddPaper`).
3. **Commit** your Changes (`git commit -m 'Add SomeAwesomePaper'`).
4. **Push** to the Branch (`git push origin feature/AddPaper`).
5. **Open** a Pull Request.

### Guidelines for Submissions

* **Relevance:** The paper/project should be related to **Generative Recommendation**, **Generative Retrieval**, or the application of **LLMs/Diffusion Models** in Search and Ads.
* **Format:** Please follow the existing Markdown format: `- Paper Title [[Paper]](Link_to_PDF)`.
* **Duplicates:** Before submitting, please search the repository to ensure the paper hasn't been added already.

### Areas of Interest
We are particularly looking for contributions in:
* Generative Search & Recommendation Frameworks.
* Semantic ID & Tokenization strategies.
* Reasoning-enhanced Recommendation (Chain-of-Thought).
* Industry-scale deployment reports.
* Fairness, Bias, and Safety in Generative Models.
